# PUMP Token — X1 Mainnet

AAA-level landing website for the PUMP token on X1 Blockchain.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Three.js (3D orb)
- Canvas API (hero bg, chart, tokenomics)

## Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2 — GitHub + Vercel Dashboard
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Framework: **Next.js** (auto-detected)
5. Click **Deploy**

## Local Dev
```bash
npm install
npm run dev
```
